How to Run CalCO:

Requirements:
1. Python 3.x installed on your computer (Python 3.8 or higher recommended)
2. Required Python packages:
   - Flask
   - pandas
   - numpy
   (You can install these by running: pip install flask pandas numpy)

Steps to Run:
1. Double-click run.bat
2. Wait a few seconds for the application to start
3. If you see any error messages:
   - Make sure Python is in your system PATH
   - Try running the batch file as administrator
   - Check if all required Python packages are installed

Note: When viewing the Excel template, save it to your desired location for future use.

Troubleshooting:
- If Python is not recognized: Open Command Prompt and type "python --version" to verify Python is installed
- If packages are missing: Open Command Prompt and run "pip install flask pandas numpy"
- If the application doesn't start: Try running run.bat as administrator
